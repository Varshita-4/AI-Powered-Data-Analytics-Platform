# Release Notes

## v1.0.0 - Initial Public Release

InsightFlow AI is an AI-powered automated data analysis platform that helps users clean datasets, perform exploratory data analysis, generate charts, create AI-powered business reports, and interact with data through a conversational interface.

### Features

#### Data Processing

* CSV dataset upload
* Missing value handling
* Duplicate removal
* Outlier detection using IQR
* Automated data cleaning

#### Data Analysis

* Exploratory Data Analysis (EDA)
* Automated chart generation
* Statistical summaries
* Dataset visualization

#### AI Features

* Gemini AI-powered insights
* AI business report generation
* Dataset question answering
* AI report caching

#### Persistence

* Report history
* Chat history persistence
* JSON-based storage

#### Downloads

* Cleaned dataset export
* Outlier-removed dataset export
* AI report PDF export

#### Frontend

* React-based dashboard
* Report viewer
* Dataset chat interface
* Report history page
* Responsive UI using Tailwind CSS

### Tech Stack

Backend:

* Python
* FastAPI
* Pandas
* NumPy
* Matplotlib
* Gemini AI

Frontend:

* React
* Tailwind CSS
* React Router
* Axios

### Status

Stable portfolio release ready for deployment.
